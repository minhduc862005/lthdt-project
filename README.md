# lthdt-project
# 📱 Phone Contact Management Application

## 📖 Introduction
This project is a simple Contact Management Application written in C++ using Object-Oriented Programming (OOP).

The program allows users to manage a list of contacts, including adding, deleting, searching, editing, and loading data from a file.

## 🎯 Features
- 📂 Load contacts from file (.txt)
- ➕ Add new contact
- 🔍 Search contact by name
- ❌ Delete contact
- ✏️ Edit contact information
- 📋 Display all contacts
- 🗂 Filter contacts by relationship (Family, Friend, Teacher, Colleague)
- 💾 Save contacts to file

## 🧠 Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling (ifstream, ofstream)
- STL (vector)
